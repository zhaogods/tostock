## 七、技术亮点

### 1. 高效查询
- 单次SQL聚合所有策略数据
- 避免10次独立查询
- 响应时间 < 1秒

### 2. 趋势计算算法
```python
# 比较近3天 vs 前3天
recent_3 = df.head(3)['avg_rate_10'].mean()
prev_3 = df.iloc[3:6]['avg_rate_10'].mean()
diff = recent_3 - prev_3

# 阈值 ±0.5%
if diff > 0.5: return 'up'
elif diff < -0.5: return 'down'
else: return 'flat'
```
