## 一、完成内容

### 1. 后端核心模块

#### ✅ instock/lib/data_asset_manager.py
- **数据类定义**：
  - `DataAsset`: 数据资产定义（9个字段）
  - `DataAssetStatus`: 数据资产状态（10个字段）

- **核心函数**：
  - `get_asset_record_count()`: 查询记录数
  - `run_quality_checks()`: 运行质量检查
  - `calculate_completeness()`: 计算完整度
  - `get_asset_status()`: 获取单个资产状态
  - `get_all_assets_status()`: 获取所有资产状态

#### ✅ instock/lib/data_asset_registry.py
- **已注册 9 个数据资产**：
  1. stock_daily - 个股日线行情
  2. stock_moneyflow - 个股资金流向
  3. etf_daily - ETF行情
  4. lhb - 龙虎榜
  5. blocktrade - 大宗交易
  6. dividend - 分红配送
  7. selection_data - 综合选股数据
  8. indicators - 技术指标
  9. patterns - K线形态

- **注册表函数**：
  - `get_all_assets()`: 获取所有资产
  - `get_asset(key)`: 根据key获取资产

#### ✅ instock/web/data_asset_handler.py
- **API接口**：
  - `GET /api/data-assets/status?date=YYYY-MM-DD`
  - `GET /api/data-assets/{asset_key}/detail?date=YYYY-MM-DD`

- **响应格式**：
```json
{
  "ok": true,
  "date": "2026-06-07",
  "assets": [
    {
      "key": "stock_daily",
      "name": "个股日线行情",
      "source": "tushare",
      "expected": 5500,
      "actual": 5487,
      "completeness": 0.9976,
      "quality_score": 95.2,
      "status": "healthy",
      "issues": [],
      "last_update": "2026-06-07"
    }
  ]
}
```

## 四、使用说明

### 启动服务
```bash
cd /home/docker/tostock
python instock/web/web_service.py
```

### 访问控制台
```
http://localhost:9988/instock/console
```

### 查看数据健康度
打开控制台后，在"系统任务中心"下方即可看到"📊 数据健康度"面板。
