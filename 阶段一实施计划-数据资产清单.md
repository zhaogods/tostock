### Task 1: 数据资产定义与注册表 (Day 1-2)

**工作量**: 2天  
**优先级**: P0（必须完成）

#### 1.1 创建数据资产基础模型

**文件**: `instock/lib/data_asset_manager.py`

**内容**:
```python
@dataclass
class DataAsset:
    """数据资产定义"""
    key: str              # 资产唯一标识
    name: str             # 中文名称
    source: str           # 数据源：tushare/eastmoney/akshare/cache
    table: str            # 数据库表名（如有）
    update_frequency: str # daily/realtime/quarterly/manual
    expected_count: int   # 预期记录数（用于计算完整度）
    quality_checks: list  # 质量检查规则
    description: str      # 描述
    depends_on: list      # 依赖的其他资产key
```

**输出**: 完成 `DataAsset` 和 `DataAssetStatus` 数据类定义

#### 1.2 创建数据资产注册表

**文件**: `instock/lib/data_asset_registry.py`

**内容**: 定义所有数据资产

**需要注册的数据资产**（根据现有数据表）:
1. `stock_daily` - 个股日线行情 (cn_stock_spot)
2. `stock_moneyflow` - 个股资金流向 (cn_stock_fund_flow)
3. `stock_fina_indicator` - 财务指标 (cn_stock_spot的财务字段)
4. `etf_daily` - ETF行情 (cn_etf_spot)
5. `selection_data` - 综合选股数据 (eastmoney)
6. `blocktrade` - 大宗交易 (cn_stock_blocktrade)
7. `lhb` - 龙虎榜 (cn_stock_lhb)
8. `dividend` - 分红配送 (cn_stock_dividend)
9. `hist_cache` - 历史K线缓存 (文件系统)

**示例**:
```python
ASSET_STOCK_DAILY = DataAsset(
    key='stock_daily',
    name='个股日线行情',
    source='tushare',
    table='cn_stock_spot',
    update_frequency='daily',
    expected_count=5500,
    quality_checks=[
        'new_price > 0',
        'volume >= 0',
        'total_market_cap > 0',
    ],
    description='全A股日线行情数据，包含价格、成交量、PE/PB/ROE等',
    depends_on=[],
)
```

**输出**: 完成 `DATA_ASSETS` 注册字典

#### 3.2 Alpine.js 组件逻辑

**在 console.html 底部添加**:
```html
<script>
function dataHealthPanel() {
  return {
    assets: [],
    loading: true,
    selectedDate: new Date().toISOString().split('T')[0],
    
    init() {
      this.loadAssets();
      // 每30秒刷新
      setInterval(() => this.loadAssets(), 30000);
    },
    
    async loadAssets() {
      try {
        const resp = await fetch(`/api/data-assets/status?date=${this.selectedDate}`);
        const data = await resp.json();
        if (data.ok) {
          this.assets = data.assets;
        }
      } catch (e) {
        console.error('加载数据资产失败', e);
      } finally {
        this.loading = false;
      }
    },
    
    getSourceIcon(source) {
      const icons = {
        'tushare': '📈',
        'eastmoney': '🌐',
        'akshare': '📊',
        'cache': '💾'
      };
      return icons[source] || '📁';
    },
    
    formatTime(time) {
      if (!time) return '未更新';
      return new Date(time).toLocaleTimeString('zh-CN');
    },
    
    viewDetail(assetKey) {
      // TODO: 打开详情弹窗或跳转
      console.log('查看详情', assetKey);
    }
  }
}
</script>
```
